= Dot Files

These are config files to set up a system the way I like it.

Most of it is based on Ryan Bates dot files, who I'm really thankful.
You can see more information here: http://github.com/ryanb/dotfiles


== Installation

  git clone https://github.com/carlosantoniodasilva/dotfiles.git ~/.dotfiles
  cd ~/.dotfiles
  rake install
